# 🥗 Nutrition Assistant Application

A full-stack MERN (MongoDB, Express.js, React.js, Node.js) web application that helps users track meals, analyze nutrition, and receive personalized dietary recommendations.

## 🚀 Features

- 👤 User Authentication (Login & Register)
- 🥗 Meal Tracking
- 📊 Nutrition Analysis
- 💡 Personalized Meal Recommendations
- 📈 Dashboard with Nutrition Summary
- 🔔 Notifications
- 📄 Export Nutrition Reports
- 👤 User Profile Management
- 📱 Responsive User Interface

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- Axios
- React Router

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Authentication
- JWT (JSON Web Token)

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/Nutrition_App.git
```

### Navigate to the Project

```bash
cd Nutrition_App
```

### Install Backend Dependencies

```bash
cd backend
npm install
```

### Install Frontend Dependencies

```bash
cd ../frontend
npm install
```

### Start Backend

```bash
cd ../backend
npm run dev
```

### Start Frontend

```bash
cd ../frontend
npm run dev
```

## 🔐 Environment Variables

Create a `.env` file in the backend folder.

Example:

```env
PORT=5000
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
